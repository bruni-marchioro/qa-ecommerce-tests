Funcionalidade:
Login

Critérios de Aceitação:

AC001
Usuário deve conseguir acessar o sistema com credenciais válidas.

AC002
Sistema deve impedir acesso quando a senha estiver incorreta.

AC003
Sistema deve impedir acesso quando o campo usuário estiver vazio.

AC004
Sistema deve impedir acesso quando o campo senha estiver vazio.

AC005
Sistema deve impedir acesso quando o usuário não existir.

AC006
Sistema deve impedir acesso quando usuário e senha estiverem vazios.
